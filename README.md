<h3>Installation:</h3>
To install:<br>
<b>If You have "git" installed</b>
<pre><code>~$ git clone "https://github.com/ankitvad/goto"
~$ cd goto
~$ make install</pre></code>
<b>If "git" is not present</b>
<pre><code>~$ wget -O goto.zip "https://github.com/ankitvad/goto/archive/master.zip"
~$ unzip goto.zip
~$ cd goto-master
~$ make install</pre></code>
